Boost QVM

Code Generator

While Boost QVM defines generic functions that operate on matrix and vector types of arbitrary static dimensions, it also provides a code generator that can be used to create compatible header files that define much simpler specializations of these functions for specific dimensions. This is useful during debugging since the generated code is much easier on the eye than the template metaprogramming-heavy generic implementations. It is also potentially friendlier to the optimizer.

The code generator is a command-line utility program. Its source code can be found in the libs/qvm/gen directory. It was used to generate the following headers:

  • 2D, 3D and 4D matrix operations:
    • boost/qvm/gen/m2.hpp (matrices of size 2x2, 2x1 and 1x2, included by boost/qvm/m2.hpp)
    • boost/qvm/gen/m3.hpp (matrices of size 3x3, 3x1 and 1x3, included by boost/qvm/m3.hpp)
    • boost/qvm/gen/m4.hpp (matrices of size 4x4, 4x1 and 1x4, included by boost/qvm/m4.hpp)
  • 2D, 3D and 4D vector operations:
    • boost/qvm/gen/v2.hpp (included by boost/qvm/v2.hpp)
    • boost/qvm/gen/v3.hpp (included by boost/qvm/v3.hpp)
    • boost/qvm/gen/v4.hpp (included by boost/qvm/v4.hpp)
  • 2D, 3D and 4D vector-matrix operations:
    • boost/qvm/gen/vm2.hpp (included by boost/qvm/vm2.hpp)
    • boost/qvm/gen/vm3.hpp (included by boost/qvm/vm3.hpp)
    • boost/qvm/gen/vm4.hpp (included by boost/qvm/vm4.hpp)
  • 2D, 3D and 4D vector swizzling operations:
    • boost/qvm/gen/sw2.hpp (included by boost/qvm/sw2.hpp)
    • boost/qvm/gen/sw3.hpp (included by boost/qvm/sw3.hpp)
    • boost/qvm/gen/sw4.hpp (included by boost/qvm/sw4.hpp)

Any such generated headers must be included before the corresponding generic header file is included. For example, if one creates a header boost/qvm/gen/m5.hpp, it must be included before boost/qvm/m.hpp. However, the generic headers (boost/qvm/m.hpp, boost/qvm/v.hpp, boost/qvm/vm.hpp and boost/qvm/sw.hpp) already include the generated headers from the list above, so they don't need to be included manually.
